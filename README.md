Tech: NodeJS, MongoDB
Description: Design an app to manage Tutorials

Note:
- title: string|min:3|max:100|required
- description: string|min:1|max:5000|required
- published: boolean

- Autogenerated:

-- updatedAt: datetime

APIs:

- Update a Tutorial by ID

- Get all Tutorials
